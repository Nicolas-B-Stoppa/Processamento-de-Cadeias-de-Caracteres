Projeto desenvolvido em C++ para a disciplina de Estruturas de Dados 1 (ED1) que implementa um Índice Invertido a partir do processamento de arquivos de texto.
A aplicação realiza a sanitização dos dados — incluindo a decodificação manual de caracteres UTF-8 e remoção de acentos —, mapeia as palavras únicas com suas respectivas
posições e utiliza busca binária $O(\log n)$ para permitir a localização rápida de termos com exibição dinâmica do seu contexto no texto.
